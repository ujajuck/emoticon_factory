# 로컬 멀티 에이전트 기반 이모티콘 제작 회사

이 저장소는 여러 AI 에이전트가 역할을 나누어 시장 조사부터 캐릭터 기획, 밑그림, 선화, 채색, 검수까지 수행하는 로컬 이모티콘 제작 파이프라인을 정의한다. 최종 목표는 하나의 캐릭터 정체성을 유지하면서 24~32종의 이모티콘 세트를 자동 생성하는 것이다.

## 목표

- 로컬 환경에서 실행 가능한 멀티 에이전트 제작 조직을 구성한다.
- 시장 조사, 캐릭터 기획, 색상 팔레트, 포즈, 이미지 생성, 검수 결과를 구조화된 데이터로 전달한다.
- 모든 제작 단계에서 캐릭터 설정서, 기준 이미지, 색상 팔레트, 비율 정보, 브랜드 가이드, 금지 요소를 공유한다.
- 최종 산출물로 플랫폼 제출이 가능한 이모티콘 패키지를 생성한다.

## 조직 구조

### CEO

전체 제작 프로세스를 관리한다.

- 프로젝트 생성
- 일정 관리
- 팀 간 결과물 전달
- 최종 승인

### 기획팀

| 역할 | 책임 | 주요 산출물 |
| --- | --- | --- |
| 시장조사 담당 | 최신 인기 이모티콘, 감정 표현, 타깃층, 경쟁작 분석 | 시장 조사 JSON |
| 캐릭터 기획 담당 | 캐릭터 콘셉트, 세계관, 성격, 표정 스타일 정의 | 캐릭터 콘셉트 JSON |
| 색상 팔레트 담당 | 메인, 보조, 강조 색상 선정 | 팔레트 JSON |
| 기획팀장 | 타깃층 적합성, 차별성, 감정 전달력, 팔레트 일관성 검수 | 승인/피드백 JSON |

### 디자인팀

| 역할 | 책임 | 주요 산출물 |
| --- | --- | --- |
| 포즈 및 상황 기획 담당 | 24~32개 이모티콘 상황 생성 | 감정·상황 목록 JSON |
| 밑그림 담당 | 포즈, 표정, 캐릭터 동작 생성 | 흑백 스케치, 포즈 시안 |
| 캐릭터 일관성 관리자 | 얼굴 비율, 눈 크기, 귀 크기, 몸통 비율, 꼬리 길이 검증 | 비율 측정 JSON |
| 디자인팀장 | 동일성, 감정 표현, 포즈 다양성, 브랜드 일관성 검수 | 승인/피드백 JSON |

### 채색팀

| 역할 | 책임 | 주요 산출물 |
| --- | --- | --- |
| 선화 담당 | 밑그림 정리, 외곽선 생성, 레이어 분리 | 레이어 이미지 |
| 채색 담당 | 지정 팔레트 적용, 레이어별 색상 채우기, 명암 추가 | 완성 이미지 |
| 채색 검수 담당 | 팔레트, 외곽선, 색상 오염, 배경 제거, 가독성 검수 | 채색 검수 JSON |
| 채색팀장 | 색상 통일, 캐릭터 일관성, 감정 전달력, 플랫폼 규격 검수 | 최종 승인 |

## 제작 파이프라인

```text
시장 조사
    ↓
캐릭터 기획
    ↓
색상 팔레트 선정
    ↓
기획팀장 검수
    ↓
표정·상황 생성
    ↓
밑그림 생성
    ↓
캐릭터 일관성 검증
    ↓
디자인팀장 검수
    ↓
선화 생성
    ↓
레이어 분리
    ↓
채색
    ↓
채색 검수
    ↓
채색팀장 검수
    ↓
최종 이모티콘 패키지 생성
```

## 공통 데이터 구조

모든 에이전트는 `schemas/project.schema.json`에 정의된 프로젝트 데이터를 공통 컨텍스트로 사용한다. 특히 다음 항목은 프로젝트 전체에서 불변 기준으로 공유해야 한다.

- 캐릭터 설정서
- 기준 이미지
- 색상 팔레트
- 비율 정보
- 브랜드 가이드
- 금지 요소 목록


## 운영 문서와 검증 도구

- `docs/agent-handoffs.md`는 각 에이전트가 읽을 입력, 갱신할 필드, 완료 조건, 재작업 루프를 정의한다.
- `scripts/validate_project.py`는 의존성 없이 프로젝트 JSON의 필수 필드, 이모티콘 수, 상태값, 팔레트 HEX 형식, ID 중복 여부를 점검한다.

## 추천 모델 구성

| 구분 | 추천 모델 | 담당 업무 |
| --- | --- | --- |
| 텍스트 에이전트 | Gemma 4 12B, GPT-OSS-20B | 시장 조사, 기획, 검수, 피드백 |
| 이미지 생성 모델 | FLUX, SDXL, ControlNet | 캐릭터 생성, 포즈 생성, 선화, 채색 |
| 비전 검수 모델 | Gemma 4 Vision, Qwen-VL | 캐릭터 일관성 검증, 색상 검수, 최종 품질 평가 |

## 핵심 성공 조건

가장 중요한 요소는 그림 실력보다 캐릭터 동일성 유지이다. 각 단계의 산출물은 캐릭터 설정서와 브랜드 가이드에 맞는지 검수되어야 하며, 불일치가 발견되면 이전 단계로 피드백을 전달해 재작업한다.
